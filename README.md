Twext
==============

A simple ruby on rails app that uses the twilio API to send text messages.

Installation and Usage
------------
Run Postrges Sever

Clone or download this repository and cd into project directory

```
$ rake db:create
$ rake db:migrate
$ rails server
```

In your browser visit localhost:3000
